BINDEOS

En este apartado tienes los mejores bindeos de cs2

Bindeos basicos

    // mute unmute game
    bind F1 "toggle volume"
    
    // mute unmute team
    bind F2 "toggle voice_modenable"
    
    // noclip
    bind F3 "sv_cheats 1; noclip"
    
    // tirar ultima granada
    bind F5 "sv_rethrow_last_grenade"
    
    // saltar con el rat√≥n y espacio
    bind "mwheeldown" +jump; bind "mwheelup" +jump; bind "space" +jump
    
    // quitar sangre, no funciona
    bind F cl_removedecals
    
    // acelerar tiempo
    bind F6 "host_timescale 5"
    
    // reestablecer tiempo
    bind F7 "host_timescale 1"
    
    // quitar humos 
    bind F8 "ent_fire smokegrenade_projectile kill; stopsound"

Jumpthrow

Como crear el Jumpthrow para cs2

    1) crear jumpthrow.cfg en C:\Program Files (x86)\Steam\steamapps\common\
    Counter-Strike Global Offensive\game\csgo\cfg ( ruta del juego donde lo tengas) 

    2) pegar las siguientes 4 lineas en jumpthrow.cfg y guardar
    -attack
    -attack2
    +jump
    -jump

    3) abrir juego y escribir en consola
    bind ALT "exec jumpthrow"
